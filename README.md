# Color_to_CMY
Enter a color value, in H,S,V format (a way of measuring colors), then let the computer calculate using subtractive color mixing what portions of that are C,M,or Y. This is designed for an iGEM project, and the CMY colors are based on Chromoproteins! I have not added black, white, or ultramarine yet, and the only value beeng used is H, soooooooo yeah.
